1. Como funciona a Criptografia + Firebase?

Mesmo conectando o banco de dados Firebase na nuvem, suas conversas estão seguras. Como o algoritmo de criptografia AES roda localmente no navegador, os pacotes que viajam pela internet já estão codificados. Se você ou alguém olhar no console do Firebase Firestore, verá apenas sequências incompreensíveis de letras e números.

3. Como hospedar este App de graça no Firebase Hosting?

  1. Crie uma conta gratuita no Firebase Console e crie um novo projeto.
  2. No seu painel, ative o banco de dados Cloud Firestore em modo de teste.
  3. Instale o Firebase CLI no seu terminal: npm install -g firebase-tools.
  4. Na pasta onde salvou este arquivo HTML (renomeie-o para index.html), rode: firebase login e depois firebase init.
  5. Selecione a opção Hosting, defina a sua pasta atual como pública e confirme o projeto.
  6. Para publicar seu WhatsApp clone na internet global, use: firebase deploy! O terminal gerará o link da sua hospedagem.
